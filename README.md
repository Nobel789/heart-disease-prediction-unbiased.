# heart-disease-prediction-unbiased.
Predicting heart disease using a curated and unbiased medical dataset to ensure fairness across demographic groups.
Project Overview: This project uses a dataset of 8,301 patient records to predict heart disease.

Key Features:

    Privacy check: Removal of sensitive PatientID.

    Data Quality: Verified for zero missing values across columns like Gender, Age, Race, and Insurance.

    Fairness: Focuses on unbiased data to prevent disparities in healthcare outcomes.

Technologies Used: Python, Pandas, Matplotlib, Seaborn, and Scikit-Learn.
